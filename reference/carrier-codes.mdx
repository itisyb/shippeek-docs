---
title: "Carrier codes"
description: "SCAC codes for all carriers supported by ShipPeek."
---

Use these SCAC codes in the `carriers` query parameter (rate requests) and the `carrier` query parameter (tracking requests).

## LTL carriers

| Carrier | SCAC code |
|---------|-----------|
| A&B Freight Line | `abfl` |
| A. Duie Pyle | `pyle` |
| AAA Cooper / MME | `aact` |
| ABF Freight System | `abfs` |
| CCFS | `ccfs` |
| Dohrn Transfer | `dohr` |
| Dugan Truck Line | `dugn` |
| Estes Express Lines | `exla` |
| FedEx Freight | `fxfe` |
| Magnum LTL | `mgul` |
| Old Dominion (ODFL) | `odfl` |
| R+L Carriers | `rlca` |
| Rossi Transportation | `rosi` |
| Saia LTL Freight | `saia` |
| Sutton Transport | `sutn` |
| XPO Logistics | `xpol` / `cnwy` |

## Parcel carriers

| Carrier | SCAC code |
|---------|-----------|
| UPS | `ups` / `upgf` / `upss` |
| FedEx | `fedex_parcel` / `fxfe` |

<Note>
  Carrier availability depends on your account configuration. Not all carriers may be enabled for your API key. Contact support to add carriers.
</Note>
